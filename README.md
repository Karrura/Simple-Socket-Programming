## About This Project

There are 2 main point in this project, Client and Server. This project just to try making a simple text, encode it, send to server, decode it, then output in the view and command promt.

## Client
Client was made using Controller, and define it in route or web.php. To run this client, we only need to ```php artisan serve``` then open the view in browser. U'll see the encode message there.

## Server
Server was made using Command, run the command defined in the code in command promt. And u'll see the message was decode there
